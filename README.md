# PokemonCONTROLE
 application mobile "Pokemon" développée avec Android Studio en utilisant Java, API et Fichier JSON, et Mock
application mobile "Pokemon" développée avec Android Studio en utilisant Java, API et Fichier JSON
L'application mobile "Pokemon" est une application Android qui permet aux utilisateurs de rechercher et de découvrir différents Pokémons. 
Cette application utilise l'API de Mocky pour récupérer des données de Pokémons telles que leur nom, leur type, leur description, leur image et leur capacité. 
Les données sont stockées dans un fichier JSON, 
qui est ensuite utilisé pour afficher les informations des Pokémons dans l'application.
Cette application mobile est un excellent exemple de l'utilisation de l'API et des fichiers JSON pour obtenir des données en temps réel et 
les afficher de manière organisée. Il est également un excellent projet pour les amateurs de Pokémons qui souhaitent découvrir et explorer davantage le monde des Pokémons.
